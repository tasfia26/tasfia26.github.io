Documentation for the Web Page

What is the Web Page About, What are There, and How does it Works

  • This Web Page is used to show some reaches that is generating Hydropower to generate Electricity for Duke Energy Progress Station in South Carolina.

  • There is an interactive Leaflet Map, where you can see the area contributing to the station, the reaches, and also the location of station.

  • The Web Page is fetching Streamflow Forecast from the National Water Model API and representing it in chart and table

  • You can have the forecast in three ranges - Short, Medium, and Long Range.

  • In the chart, you can see a Threshold line, which means the streamflow when exceeds the threshold line that flow can be used to generate electricity. If it doesn't, then that flow can not be used for electricity generation.

  • You can also look into the benefits of using the streamflow for electricity generation for Duke Energy Progress Station at the beginning of the Web Page and have an overview.

  • Also, I have added my personal information at the left of the Web Page.

    • How to Use the Web Page

  • In the web page, you can see the Leaflet Map. There is a Polygon which is the boundary for the Duke Progress Station marked in Grey. The blue line shows the reaches contributing to the electricity generation, and the red point is the location of Duke Energy Progress Station.

  • Under the Map, you can see a Toggle Layer where there is the Reach Layer, and the Duke Energy Progress Station Layer. You can check or uncheck the layers to be shown on the map.

  • If the Reach Layer is unchecked, you can see the individual reach on map, when you click on the Reach ID's that are present under the Toggle Layer.

  • If you "Click" a Reach ID, you can see a Table and a Chart having the Streamflow Forecast and the Threshold, and you can see the Reach on the Map.

  • There is a Drop Down Box from where you can Select the Range of the Forecast.
